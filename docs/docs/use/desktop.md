# Desktop app

Anton is also available as a desktop application. It wraps the same engine
as the terminal interface — the same provider setup, memory, skills, and
workspaces — behind a graphical UI.

## Download

- **macOS**: [anton-latest.pkg](https://downloads.mindsdb.com/anton/mac/anton-latest.pkg)
- **Windows**: [anton-latest.exe](https://downloads.mindsdb.com/anton/windows/anton-latest.exe)

Run the installer and launch Anton like any other app. On first launch you
pick an LLM provider, just like the terminal flow — see
[Pick a provider](/start/pick-a-provider).

:::note
The desktop app lives in the separate Cowork repository, and detailed desktop
documentation is coming there. This page will link to it once published.
:::

In the meantime, almost everything in these docs applies to the desktop app
too, since both interfaces drive the same engine. Start with
[How a turn works](/use/chat-basics).
